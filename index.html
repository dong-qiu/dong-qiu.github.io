I am a senior software engineer at System Engineering Department, R&D Comptence Center, 2012 Lab, Huawei Technologies, working on how to improve software development productivity.
I obtained my PhD in 2015 at Southeast University, advised by Prof. Bixin Li. My research interests span software analysis and testing, regression testing and monitoring in web services & SOA, database applications and programming languages.
